About

A browser-based iteration of a manual pipeline.

Builds a mass booklet for the Extraordinary Form (1962 Missal). Select a mass from the calendar, configure the ordinary settings, and export.

Using the editor

Choose a mass
Pick Tempora or Sancti, then select a mass. The ordo loads automatically with propers, ordinary, and chants pre-filled.
Ordinary & creed
Use the dropdowns to select a chant setting for the ordinary and creed. Language can be changed independently.
Add / reorder items
Click a + zone to insert a rubric, text, chant, section title, or ordinary block. Drag the ⠿ grip to reorder.
Search for chants
In the Add form, switch to Search and type a GregoBase ID, URL, or opening words.
Edit GABC
Click the pencil button on any chant to edit the GABC notation directly.
Missalette / Pew Sheet
Each item has M and P buttons to control which document it appears in.
Print
Opens the browser print dialog. Chants are re-rendered at A5 column width before printing.
Download PDF
Sends the mass to a cloud builder (LuaLaTeX) and downloads the result.
Undo / Redo
The ↶ / ↷ buttons (or Ctrl+Z / Ctrl+Y) undo and redo item deletions.

Sources

Chant notation
GABC files from GregoBase.
Latin texts & translations
From the Divinum Officium project.

Limitations

No persistent storage
Edits are lost when you close the tab. Export before leaving.
Snapshot data
DO and GregoBase data is periodically regenerated, not live.
Chant matching
Chant notation is automatically matched to DO text. Masses in PDF releases have been verified, but other masses may have errors.
